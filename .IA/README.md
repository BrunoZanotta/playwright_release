# .IA

Esta pasta guarda ativos de IA versionaveis do projeto: agents, workflows, decisoes tecnicas e notas de evolucao.

Nao incluir:

- secrets, tokens ou credenciais privadas
- traces, videos, screenshots ou reports pesados
- estado local de ferramenta
- arquivos gerados que dependam da maquina do usuario

Os arquivos executaveis dos Playwright Test Agents ficam no caminho esperado pelo cliente (`.github/chatmodes` e `.vscode/mcp.json`). Esta pasta documenta como eles devem ser mantidos.
